# FraudShield AI - Anti-Spoofing Demo

FraudShield AI is a modern web demonstration of a fraud detection engine designed for delivery platforms. It simulates real-time behavioral analysis to identify GPS spoofing, device manipulation, and coordinated fraud attacks.

## 🚀 Features

- **Advanced Logic Engine**: Calculates risk based on GPS-IP correlation, movement patterns, and hardware integrity.
- **Trust Score Tracking**: Visualizes system confidence using dynamic circular progress indicators.
- **Behavior Profiling**: Categorizes user behavior into profiles like "Verified Hardware", "Steady Movement", etc.
- **Fraud Network Alerts**: Detects and alerts on coordinated clusters (Shared IP + high risk).
- **Premium UI**: Built with a sleek glassmorphism aesthetic, Lucide icons, and Outfit typography.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern styling with glassmorphism and animations.
- **JavaScript**: Core fraud detection logic and UI interactivity.
- **Lucide Icons**: Premium iconography.

## 📂 Project Structure

- `index.html`: The main dashboard structure.
- `style.css`: All premium styling and animations.
- `script.js`: Behavioral analysis and risk calculation logic.

## 🚦 How to Run

Simply open `index.html` in any modern web browser to start the simulation.
